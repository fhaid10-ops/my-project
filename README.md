# المصاريف اليومية | Daily Expenses

صفحة ويب أنيقة وبسيطة لتسجيل المصاريف اليومية باللغة العربية مع دعم اتجاه RTL. تتضمن خانة خاصة لتبديل زيت السيارة تسجّل نوع السيارة وقراءة عدّاد الكيلومترات.

## المميزات

- إضافة مصروف مع: المبلغ، نوع الشراء، التاريخ، الساعة.
- مفتاح تبديل خاص "تبديل الزيت" يظهر حقول إضافية:
  - نوع السيارة.
  - قراءة العدّاد بالكيلومتر.
- عرض آخر تبديل زيت مسجّل لتذكّرك متى تحتاج للتبديل التالي.
- إحصائيات فورية: مجموع اليوم، الشهر، الكلي، وعدد العمليات.
- بحث وفلترة حسب نوع الشراء.
- تصدير كل المصاريف كملف CSV.
- حفظ البيانات محلياً في المتصفح (localStorage) — لا حاجة لخادم.
- تصميم عصري ومتجاوب مع جميع الأحجام (جوال / تابلت / حاسوب).

## طريقة التشغيل

لا يحتاج المشروع إلى تبعيات أو أدوات بناء. فقط افتح الملف `index.html` في متصفحك.

أو شغّل خادماً محلياً بسيطاً:

```bash
python3 -m http.server 8000
```

ثم افتح: <http://localhost:8000>

## الملفات

- `index.html` — بنية الصفحة.
- `styles.css` — التنسيقات والتصميم.
- `app.js` — منطق التطبيق وحفظ البيانات.
